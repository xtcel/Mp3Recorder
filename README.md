# Windows MP3录音程序
这是一款window平台下的MP3录音控制台程序，使用lame库编码mp3格式。

# 使用
编译打包对应的平台，x64 或者 x86
使用控制台运行：
```
./mp3_stream.exe -t=5 "-fn=c:\\Users\\CaiDan\\Desktop\\workspace\\test.mp3"
```
-t 是录音时间，默认 10s
-fn 是录音文件存放目录，默认 当前目录下
